17  /*! \class FileInfo
18  * \author Greg Sjaardema
19  * \brief Return information about the specified file.
20  *
21  * A very minimal class (at least it used to be) for providing
22  * information about a file. FileInfo provides information about a
23  * file's name, path, and type (directory, symbolic link, file). Other
24  * information could be added as needed. It currently does not cache
25  * any information, so if it is heavily used, a caching capability
